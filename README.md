# 🥚 Yoshi Eggs Store 3DS

Loja de homebrew BR para Nintendo 3DS 🇧🇷

## 📲 Como usar

1. Abra o Universal Updater
2. Vá em "Settings"
3. Clique em "Select UniStore"
4. Adicione a URL:

https://raw.githubusercontent.com/SEUUSER/yoshi-eggs-store/main/apps.json
